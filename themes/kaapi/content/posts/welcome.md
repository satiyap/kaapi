Welcome to Kaapi, a minimal and responsive Hugo theme inspired by filter coffee—simple, effective, and focused on what matters.

## Features

Kaapi focuses on what matters most: your content. With a clean, distraction-free layout, your readers can focus on your writing.

### Clean Typography

The theme uses system fonts for fast loading and excellent readability. Font sizes are optimized for comfortable reading on all devices.

### Responsive Design

Kaapi looks great on all screen sizes, from mobile phones to large desktop displays. The layout adapts seamlessly to provide the best reading experience.

### Lightweight

No JavaScript dependencies means fast page loads and excellent performance. The theme is built with pure HTML and CSS.

## Getting Started

To create a new post:

```bash
hugo new posts/my-post.md
```

Then edit the markdown file in `content/posts/` and start writing!

## Customization

You can customize the theme by:

- Editing `hugo.toml` for site settings
- Overriding templates in your site's `layouts/` directory
- Adding custom CSS in `assets/css/`

Happy writing!
