# Filter Your Thoughts

**Kaapi** is a Hugo theme inspired by the focused character of traditional filter coffee. Like coffee brewed slowly through a filter, this theme emphasizes patience and clarity in design.

## Design Philosophy

Filter coffee is about simplicity and quality. This theme follows that approach with clean typography, thoughtful contrast, and uncluttered layouts. Readable backgrounds with well-chosen accent colors create a comfortable reading experience.

## Key Features

- **☕ Coffee-Inspired Palette** - Brown accent colors (#8b4513, #6b3410) with good contrast
- **✨ Subtle Animation** - Post title underlines on hover
- **📱 Responsive** - Works well on all device sizes
- **⚡ Fast Loading** - Minimal CSS, no JavaScript dependencies
- **📖 Clear Typography** - Readable fonts and spacing
- **💻 Code Highlighting** - Syntax highlighting support
- **📑 Table of Contents** - Optional TOC for longer posts
- **🔗 Clean Navigation** - Simple header with About link
